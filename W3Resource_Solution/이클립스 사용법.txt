객체: 작용(어떠한 동작을)할 수 있는 물체
	그래서 서로서로 작용이 가능하면 모두 객체

객체지향: 모든 것을 이 객체로 만들고자 지향함

객체: 행동(behavior)과 상태(status)를 가지고 있다 by java 개발자
ex)	상태-이름을 가지고 있다
	행동-움직인다

ex)자전거를 객체라고 보자. 기어, 속도 등은 상태, 기어를 올린다 등은 행동



그래서 객체를 어떻게 표현할 것인가? -> 클래스가 필요

클래스는 설계도/프로토타입이다. 그래서 클래스로부터 객체를 만들어 낼 수 있다

어떠한 클래스로부터 new라는 키워드를 통해서 객체를 생성한다

==>설계는 클래스에 하고 실제로 사용하는 물품은 그로부터 만들어낸 객체

그렇다면 클래스에 객체의 행동과 상태는 어떻게 표현해 놓는가?

==>행동은 method로 상태는 field로 클래스에 표현


field
"클래스 안에 선언해 놓은 변수"로 "객체가 어떠한 상태"를 가진다는 것을 나타내기 위함
객체의 상태를 나타내는 것



method
객체가 어떤 행동을 한다.



클래스명 변수명 = new 생성자();
클래스이름이 데이터타입처럼 취급이 된다. 
변수명이라는 변수는 "클래스명으로부터 만들어진 객체"를 가리킨다는 뜻이다.
참조변수는 stack에 저장이 되고, 실제 객체는 heap 영역에 생성이 된다
